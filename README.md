## Setup

  ##### Clone the repository
```bash
git clone https://github.com/ishandeveloper/tic-tac-toe.git
```
  ##### Move to the desired folder
```bash
cd \tic-tac-toe
```
  ##### To install the dependencies, simply write
```bash
yarn install
```

  ##### To run the app, simply write
```bash
yarn start
```
